# Java 代码中的注释规范

## 一、成员变量注释、类注释、方法注释类型规则

### 1、场景

在需要写 `/** */` 注释的场景，请遵循我的规则，场景包括：

- 成员变量注释
- 类注释
- 方法注释

行注释 `//` 不遵循此规则，另外普通的块注释 `/* */` 也不遵循此规则。

遵守此规范的注释：

```java
/**
 * ......
 */
```

不遵守此规则的块注释：

```java
/*
......
 */
```

不遵守此规则的行注释：

```java
// ......
```

### 2、规则描述

在成员变量注释、类注释和方法注释上，每一个段落前请使用 `<p>` 标签，另外如果注释中有无序列表或有序列表，请使用 `<pre></pre>` 标签。

### 3、规则示例

```java
/**
 * <p>段落一
 *
 * <p>段落二
 *
 * <p>无序列表
 * <pre>
 * - 无序列表 第一条
 * - 无序列表 第二条
 * - 无序列表 第三条
 * </pre>
 *
 * <p>有序列表
 * <pre>
 * - 有序列表 第一条
 * - 有序列表 第二条
 * - 有序列表 第三条
 * </pre>
 *
 */
```

## 二、块标签

```java
/**
 * <p>
 * 
 * @author z2huo
 */
```

生成的 Java 类中的块标签要有：

- `@author`，作者信息，值为 `z2huo`

## 三、注释内容

### 1、行末句号

注释内容如果只有一行的话，后面不要加句号。

正例：

```
/**  
 * <p>名称
 */  
private String name;

/**  
 * <p>Amount, in cents
 */  
private Long amountInCents;
```

反例：

```
/**  
 * <p>载荷名称。
 */  
private String name;

/**  
 * <p>Amount, in cents.
 */  
private Long amountInCents;
```
